# CLAUDE.md — Визитка Ph.Dr. Olga Tureva

## Что это

Одностраничная персональная цифровая визитка кризисного психолога.
Весь проект — два файла: `index.html` и `photo.jpg`. Никаких сборщиков, фреймворков и зависимостей.

## Правила работы с кодом

- **Не добавлять** фреймворки, npm, bundler — проект намеренно zero-dependency
- **Не разбивать** на отдельные CSS/JS файлы — всё хранится в одном `index.html`
- **Не менять** шрифт (Palatino/Georgia) и цветовую схему без явной просьбы владельца
- При добавлении новых иконок социальных сетей — использовать inline SVG, как у существующих

## Структура index.html

```
<head>
  <style>  ← весь CSS здесь, ~280 строк
</head>
<body>
  <canvas id="stars">        ← анимированный фон
  <div class="card">
    .avatar                  ← фото + вращающийся золотой ореол
    h1                       ← имя с gold-shimmer анимацией
    .subtitle1 / .subtitle2  ← должность и специализация
    .divider                 ← декоративный разделитель с ✦
    .socials                 ← кнопки: .tg .wa .web .em
    .cta                     ← главная кнопка «Написать мне»
    .contact-info            ← текстовые ссылки внизу
  </div>
  <script>  ← Canvas-анимация звёзд и падающих звёзд
</body>
```

## Canvas-анимация (JS)

- `STAR_COUNT = 220` — количество звёзд
- Звёзды трёх оттенков: золотые / лавандовые / белые
- Падающие звёзды: интервал 2.5–7 сек, хвост до 200px
- Функция `drawFull()` — основной render loop через `requestAnimationFrame`
