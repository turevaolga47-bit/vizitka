## Структура проекта

```
vizitka/
├── index.html   # Единственная страница — вся логика, стили и анимация
└── photo.jpg    # Фото владельца карточки
```

## Технологии

- Чистый HTML/CSS/JS, без фреймворков и зависимостей
- Canvas API — анимированное звёздное небо с падающими звёздами
- CSS-анимации — золотой shimmer заголовка, вращающийся ореол аватара, pulse-ring у кнопки
- `backdrop-filter: blur` — стеклянный эффект карточки

## Дизайн-система

- Фон: `#07001a` (глубокий тёмно-фиолетовый / космос)
- Акцент: золото `#c8922a → #f5d060 → #e8b830`
- Текст: `#f1e8d0` (тёплый кремовый)
- Карточка: `rgba(15, 5, 40, 0.72)` с `backdrop-filter: blur(18px)`
- Шрифт: Palatino / Georgia (serif)
