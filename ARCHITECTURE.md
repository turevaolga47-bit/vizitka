# Архитектура визитки психолога
## Как штамповать новые визитки за 10 минут

---

## Файловая структура

```
папка клиента/
├── index.html      ← берётся из template.html (переименовать)
├── photo.jpg       ← фото специалиста (любой формат, переименовать в photo.jpg)
└── socials.txt     ← опционально: текстовый список всех ссылок
```

---

## Как создать новую визитку

### Шаг 1 — Скопировать шаблон
```
template.html  →  скопировать  →  index.html  (в новую папку клиента)
```

### Шаг 2 — Добавить фото
Положить фото рядом с index.html, назвать `photo.jpg`

### Шаг 3 — Заполнить CONFIG в начале файла
Открыть index.html в любом редакторе (VS Code, Notepad++).
Найти блок `const CONFIG = {` — он в самом начале, до `</script>`.
Заполнить только этот блок — больше ничего не трогать.

### Шаг 4 — Открыть в браузере и проверить
Дважды кликнуть на index.html → открывается в браузере.

### Шаг 5 — Залить на GitHub Pages
1. Создать репозиторий на github.com (под аккаунтом клиента или общим)
2. Загрузить index.html + photo.jpg
3. Settings → Pages → Source: main branch → Save
4. Ссылка: `https://USERNAME.github.io/REPO-NAME/`

## Блоки CONFIG — что заполнять

| Поле | Что писать | Обязательно |
|------|-----------|-------------|
| `name` | Имя и фамилия | ✅ |
| `degree` | Степень/звание, или `''` | |
| `photo` | `'photo.jpg'` или `''` для инициалов | |
| `chips` | Бейджи под именем (до 6 штук) | ✅ |
| `bio` | 2-3 строки о себе | ✅ |
| `links` | Все соцсети (пустые ставь `''`) | ✅ |
| `about` | Текст блока "О мне" | ✅ |
| `stats` | 3 цифры-показателя | ✅ |
| `services` | Список услуг (до 6) | ✅ |
| `promo` | Промо-блок (или `null`) | |

---

## Цвета чипов

| `style` | Цвет |
|---------|------|
| `'gold'` | Золотой |
| `'pink'` | Розовый |
| `'teal'` | Бирюзовый |
| `''` или не указывать | Серый (нейтральный) |

---

```

'#f59e0b'  — золотой
```

---

## Кнопки CTA (главные)

Показываются автоматически если ссылка заполнена:
- `links.telegram` → синяя кнопка «Написать в Telegram»
- `links.max`      → синяя кнопка «Написать в MAX»
- `links.whatsapp` → зелёная кнопка «Написать в WhatsApp»

---

## Соцсети — что скрывается автоматически

Если поле пустое (`''`) — кнопка этой соцсети не показывается.
Не нужно удалять строки из конфига — просто оставь `''`.

| Этап | Время |
|------|-------|

| Добавить фото | 1 мин |

| **Итого** | **~12 минут** |

---

## Технические детали (для разработчика)

- **Нет зависимостей** — чистый HTML/CSS/JS, работает без интернета
- **Один файл** — всё включено (стили, скрипты, иконки SVG)
- **Анимации** — CSS keyframes + Canvas API (звёзды, метеориты, частицы)
- **Адаптив** — mobile-first, корректно на экранах от 320px
- **Производительность** — только `transform` и `opacity` в анимациях (GPU)
- **Фото-фоллбек** — если photo.jpg нет, показывает инициалы

## Структура блоков визитки (сверху вниз)

```
1. HERO          ← фото + имя + степень + чипы + bio
2. CTA КНОПКИ   ← Telegram / MAX / WhatsApp (по наличию)
3. СОЦСЕТИ       ← сетка 5 колонок, иконки с подсветкой
4. О МНЕ         ← текст + 3 статистики
5. УСЛУГИ        ← список с цветными точками
6. КОНТАКТЫ      ← строки с иконками и стрелками
7. ПРОМО         ← опциональный блок (ретрит / курс / событие)
```
